# 版本更新日志

## v1.0.0 (2024-01-XX)

### 首个正式版本

**核心功能**
- ✅ 文章管理系统（创建、编辑、删除、发布）
- ✅ Markdown 编辑器（SimpleMDE）
- ✅ 分类管理
- ✅ 评论系统（需审核）
- ✅ 管理员认证（NextAuth.js + JWT）
- ✅ 仪表板统计
- ✅ 响应式设计

**技术栈**
- Next.js 14 (App Router)
- TypeScript
- MySQL
- Tailwind CSS
- NextAuth.js

**安全特性**
- 密码 bcrypt 加密
- JWT Session
- API 权限验证
- SQL 参数化查询
- 评论审核机制

**已知限制**
- 仅支持单管理员
- 无图片上传功能
- 无文章搜索
- 无标签系统
